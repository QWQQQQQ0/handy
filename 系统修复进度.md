### 8. ✅ 修复 C 盘根目录 ACL 权限 — 文件资源管理器恢复访问

**问题**：chkdsk 恢复文件时损坏了 C:\ 的安全描述符，导致 `BUILTIN\Users` 和 `Authenticated Users` 的读取权限丢失。普通用户（非提权）在文件资源管理器中点击 C 盘提示"拒绝访问"。

**修复**：
```batch
icacls C:\ /grant "BUILTIN\Users:(RX)"
icacls C:\ /grant "NT AUTHORITY\Authenticated Users:(RX)"
```

**验证**：文件资源管理器 C 盘访问正常 ✅

虽然当前一切正常，但**间歇性磁盘错误可能复发**，建议：

1. **及时备份重要数据** — 间歇性 SSD 错误是硬件退化的信号

3. 使用 CrystalDiskInfo 或 `wmic diskdrive get status` 持续监控 S.M.A.R.T.
4. 如再次出现 `STATUS_IN_PAGE_ERROR`（0xc0000006），考虑更换 SSD
